◐ Library of Aletheia ◑

Every page already exists. Find the address.

The library contains every possible book of 1,312,000 characters drawn from the 42-character alphabet a-z, comma, period, exclamation, question mark, hyphen, space, and digits 0-9. Each book lives at a unique address of the form room.wall.shelf.book.page. Searching text doesn't generate a new page — it computes which page already contains it.
